# adobe jsx

command line tool to execute adobe extendscript .jsx file

## Install

```sh
# npm
npm i -g adobe-jsx

# pnpm
pnpm i -g adobe-jsx
```

## Usage

After installation, you can use command `jsx` to execute the jsx script.

```sh
jsx /path/to/jsxScript.jsx
```
